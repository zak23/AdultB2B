# Roadmap (Derived From PRD)

This roadmap reflects the PRD in `AdultB2B_PRD.md` (last updated Jan 31, 2026). It does not claim implementation status.

## V1 Core Features

- Authentication and RBAC
- Personal and company profiles
- Networking: follows, connections
- Feed with posts (text, image, video, links)
- Engagement: reactions, comments, reposts
- Messaging: 1:1 and group
- Groups and community features
- Events and RSVPs
- Basic analytics
- AI assist (light) and moderation (optional)

## V1.5+

- Engagement-weighted feed
- Stronger analytics and insights
- Scheduling, drafts, content queue

## Phase 2

- OAuth sign-in
- Ads and sponsored content
- Enhanced targeting and performance tracking

## Recommended Tracking

Tie roadmap milestones to `docs/project-status.md` so new team members can see what is planned vs what is actually implemented.
